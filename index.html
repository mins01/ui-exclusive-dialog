UI modalDialog

  • dialog 태그를 사용하여 modal(뒷 부분의 사용할 수 없게 함) 을 구성한다.
  • alert, confirm, prompt 를 기본 제공한다
  • form의 유효성 검사 이슈
  • HTML 스펙상 : method="dialog"은 constraint validation을 트리거하지 않도록 명시됨 ( https://html.spec.whatwg.org/multipage/form-control-infrastructure.html#form-submission-algorithm )
  • 실제 브라우저 (Chrome 등) : dialog.showModal() 상태에서 button type="submit" 클릭 시 validation 수행함
